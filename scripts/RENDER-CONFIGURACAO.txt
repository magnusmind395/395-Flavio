RENDER — CONFIGURAÇÃO OBRIGATÓRIA (copie exato)
================================================

OPÇÃO 1 (recomendada)
  Root Directory:     server
  Build Command:      npm install --include=dev && npm run build
  Start Command:      npm start
  Health Check Path:  /api/health

OPÇÃO 2 (se Root Directory estiver VAZIO / raiz do repo)
  Root Directory:     (vazio)
  Build Command:      npm run render-build
  Start Command:      npm run render-start
  Health Check Path:  /api/health

ERRADO (causa "Cannot GET /api/health"):
  Root vazio + Build "npm run build" + Start "npm start" ou preview
  → sobe o Vite/frontend, NÃO a API

Depois: Manual Deploy → teste https://three95-flavio.onrender.com/api/health
Deve retornar JSON {"status":"ok",...} — NÃO HTML "Cannot GET"
